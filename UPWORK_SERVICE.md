## Project Description (Main)

### "Building AuditWave: A Production-Grade Website Audit Platform"

I recently completed **AuditWave**, a comprehensive all-in-one website audit tool that showcases advanced full-stack development capabilities. This project demonstrates expertise across the entire web development stack — from server-side browser automation to responsive frontend architecture.

#### What is AuditWave?

AuditWave enables users to scan any URL for:
- ✅ WCAG 2.2 accessibility violations (axe-core engine)
- ✅ Core Web Vitals performance metrics (Lighthouse integration)
- ✅ SEO issues (meta tags, structured data, heading hierarchy)
- ✅ JavaScript/CSS bundle analysis & technology detection
- ✅ Next.js-specific rendering & optimization analysis
- ✅ A/B performance comparisons between URLs
- ✅ Multi-device testing (4 responsive viewports)
- ✅ PDF report generation

All completely free, no login required, 100% client-side data (localStorage).
